# Suhang2222
이 프로그램은 이미지의 ai 생성 여부를 보여주는 기능을 담고있다.

최근 인스타그램이나 틱톡, 유튜브등 많은 소셜미디어 매체에는 ai로 만들어진 영상과, ai로 만들어진 사진이 많아지고 있다. 
이 이미지들은 사람들의 이해를 돕기위한 유용한 측면에서 활용될 수 있지만, 반대로 누구나 쉽게 만들 수 있기 때문에 사람들을 속이기 위한 나쁜 측면으로 악용될 수도 있다. 
이미지 생성 ai가 나날이 발전 되어가는 추세이므로 ai로 인한 조작 정보가 사실과 구분할 수 없는 수준으로 발전한다면, 진실이라는 단어가 무의미해진 미래가 올 것이다.
따라서 나는 ai 이미지 판별 프로그램으로 진실이라는 단어를 지키기 위한 프로젝트의 시작을 끊고 싶다.
